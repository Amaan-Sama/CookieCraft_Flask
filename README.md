# Flask_Project — CookieCraft 

CookieCraft  is a full-stack web application developed using Flask, SQLAlchemy, Bootstrap, and SQLite. The project allows users to create, manage, update, and delete cookie recipes through an interactive and responsive web interface.

A key feature of the project is dynamic image generation, where recipe descriptions are converted into visually generated food images and displayed automatically inside the application.

## Features

* Add new cookie recipes
* Update existing recipes
* Delete recipes
* Dynamic image generation from recipe descriptions
* Automatic image saving and rendering
* Responsive UI using Bootstrap and custom CSS
* SQLite database integration using SQLAlchemy

## Technologies Used

* Python
* Flask
* SQLAlchemy
* SQLite
* Bootstrap
* HTML/CSS
* REST API Integration

## Important Note

To use the image generation functionality, users may need to provide their own API key depending on the image generation service being used.

## Project Goal

The main goal of this project was to understand:

* full-stack web development
* Flask backend architecture
* database integration
* API communication
* dynamic file handling
* frontend and backend connectivity
